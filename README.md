# crawlers
# 一个简单的爬虫测试
本站是个人出于兴趣，用Node.js对CNodejs中文社区网站首页（ http://cnodejs.org ）的版区信息进行实时抓取，然后重新展示于该页面。

## 后台
+ Node的http爬取目标数据，express设置静态资源及监听端口
+ npm进行依赖管理
  <pre><code>npm install</pre></code>

## 前端
+ Vue.js进行部署及数据绑定，axios前后数据交换
+ Bootstrap简单搭建页面
+ bower进行依赖管理
  <pre><code>bower install</pre></code>
